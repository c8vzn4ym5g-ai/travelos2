# TravelOS PRD

TravelOS is a private-first personal travel system for recording trips, photos, videos, costs, hotels, restaurants, transportation, places, and reflections. It should feel like a long-term memory platform rather than a simple blog.

## Goals

- Record every trip with structured details and narrative notes.
- Search travel memories by country, city, date, people, cost, rating, media, and future natural-language questions.
- Provide map, timeline, album, and analytics views that remain useful after hundreds of trips.
- Support private use first, with selective public sharing later.

## MVP Scope

- Responsive application shell
- Trip list, trip detail, and trip creation flow
- Country and city organization
- Journal entries with rich text-ready fields
- Photo metadata model and album placeholders
- Cost tracking model
- Hotel, restaurant, transportation, and attraction models
- Map-ready coordinates on places and trips
- AI-ready normalized content fields
- Documentation and task workflow for Codex

## Deferred Scope

Authentication, media uploads, OCR receipts, real map provider integration, AI search, public sharing, PDF export, and collaboration are deferred until the MVP foundation is stable.

## Acceptance

The MVP is acceptable when a user can create a trip, view it in a list, open details, attach structured places and costs, and see the foundation for map, album, timeline, and AI search features.
